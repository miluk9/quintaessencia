# Lembretes Ativos

Este arquivo é lido automaticamente no início de cada conversa.
Contém avisos, alertas e coisas que a Michelle quer manter no radar.

---

## Como funciona

- Adicionar um lembrete com `/lembrar [o que lembrar]`
- Remover um lembrete com `/lembrar remover [título]`
- Claude lê este arquivo no início de cada sessão e avisa sobre lembretes ativos
- Lembretes com data são apontados quando a data se aproximar
